# Next.js Project Template

モダンな技術スタックを使用したNext.jsプロジェクトテンプレート

## 📋 前提条件

- Node.js 24.5.0
- pnpm latest
- Docker (オプション、コンテナ開発用)

このプロジェクトは複数のバージョンマネージャーをサポートしています：

## 🚀 クイックスタート

### 1. セットアップ

```bash
# mise使用時
mise run setup

# または手動セットアップ
pnpm install
cp .env.example .env
```

### 2. 開発サーバー起動

```bash
# mise使用時
mise run dev

# または
pnpm dev
```

http://localhost:3000 でアプリケーションが起動します
