# Favicon Placeholder

This file serves as a placeholder for favicon.ico.

To add a favicon:
1. Replace this file with a favicon.ico file
2. Ensure the favicon is 16x16 or 32x32 pixels
3. Use ICO format for best compatibility

Alternative locations for favicons:
- /favicon.ico (root level)
- /public/favicon.ico
- /public/icons/ (for various sizes)

Recommended favicon formats:
- favicon.ico (16x16, 32x32)
- apple-touch-icon.png (180x180)
- icon-192.png (192x192)
- icon-512.png (512x512)
